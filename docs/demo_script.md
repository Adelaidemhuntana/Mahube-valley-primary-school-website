# Demo Script

Use this script when presenting.

## Opening

My project is called Smart School Placement Hub.

It is an AI powered data platform for school capacity planning, admissions analytics and learner placement.

The goal is to show how school applications can become useful data for parents, schools, districts and the department.

## Step 1

I will first show the list of schools.

This shows the operational capacity data.

## Step 2

I will submit a parent application.

The learner is applying for Grade 1 at Mahlasedi Masana Primary School.

The learner lives 3.2 km from the school, so the learner is within the 5 km rule.

## Step 3

The system creates a five day offer.

This means the seat is not taken yet. It is under offer.

## Step 4

I will show the school capacity again.

Available seats went down.

Under offer seats went up.

## Step 5

Now I will accept the offer.

The seat moves from under offer to taken.

## Step 6

I will show the analytics summary.

This is where the data engineering part starts.

The same data can later move into a data lake, ETL pipeline, data warehouse and dashboard.

## Closing

This project is different from a basic application because it shows a full data flow.

It captures events, updates capacity, stores data and prepares analytics for decision making.
